Technical details

Why does it look so basic?

It's all about the content.

The CSS has been handcrafted for maximum legibility. It uses clear, high contrast black and white colours with the minimum use of blue required to distinguish links. Bold is used to draw attention, and spacing is used to allow for easy flow.

It works well on most screen orientations, resolutions, brightness levels and also works well with text based browsers and screen readers, as well as automated crawlers.

How is it so fast?

The content is completely handcrafted and uses only simple HTML. It is very small, the index page is only 880 bytes, and has very high cache timeout.
